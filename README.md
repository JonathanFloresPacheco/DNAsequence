# DNAsequence
 Node.js with Sequelize in DNA sequence
### Requirements

`create database in mysql with name dnasequence`
`CREATE DATABASE dnasequence;`
`add .env of file is env example`
`changed dates username and password of data base`

### Project setup

`npm install`
